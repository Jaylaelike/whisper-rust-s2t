üìã API Endpoints

POST /transcribe

Upload an audio file and get transcription results with optional risk analysis

Parameters:
  • audio - Audio file (multipart/form-data)
  • language - Language code (query parameter, default: "th")
  • backend - Processing backend: "cpu", "gpu", "coreml" (query parameter, default: "cpu")
  • chunking - Enable chunking for long audio (query parameter, default: "true")
  • risk_analysis - Enable content risk detection (query parameter, default: "false")
POST /risk-analysis

Analyze text content for risk detection (gambling, illegal activities, etc.)

Body:
  • {"text": "content to analyze"} - JSON with text to analyze
GET /health

Check API health status

GET /languages

Get list of supported languages
